水平垂直居中有好多种实现方式，主要就分为两类不定宽高和定宽高
以在body下插入一个div为例
1. 定宽高
使用定位+margin
2. 使用定位+transfrom
3. 不定宽高
不定宽高的方法基本都适用于定宽高的情况
这里把div的宽高按照内容展开
使用定位+transform同样是适用的

还有一些其他的方法比如使用父容器使用flex，grid，table
